# 插画视觉还原测试用例（手工）

## 目标
- 插画增强调性但不影响可用性
- 不遮挡任何主要交互（表格、按钮、分页、弹窗等）
- 在不同分辨率下保持位置合理与缩放自然

## 用例 1：大屏（≥ 1440px 宽）
1. 打开“公司文件柜”页面
2. 检查右侧空白区域是否出现插画
3. 检查表格区域、分页、批量条、浮动详情是否不被遮挡
4. 滚动页面：插画应固定在视口右下（不随内容滚动）

## 用例 2：中屏（1100–1440px）
1. 打开“公司文件柜”页面
2. 插画应缩小并保持在右下角，透明度不影响文字可读性

## 用例 3：小屏（≤ 1100px）
1. 打开“公司文件柜”页面
2. 插画应隐藏（避免挤占空间）

## 用例 4：无障碍与交互
1. 使用 Tab 键遍历页面：插画不应获得焦点
2. 使用屏幕阅读器：插画应被忽略（aria-hidden）
3. 点击/拖拽：插画不应拦截鼠标事件（pointer-events: none）

## 用例 5：动效偏好
1. 系统开启“减少动态效果”
2. 刷新页面
3. 插画不应再出现漂浮动画

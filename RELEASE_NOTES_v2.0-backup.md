# V2.0 backup

标签：v2.0-backup

## 主要更新

- 文件柜
  - 支持拖拽将“已注册文件”移动到左侧分类树（含拖拽提示、落点高亮、完成提示）
  - 支持“未分类”固定入口与一键自动分类（关键词规则）
  - 显示入库时间（registered_at），并按入库时间排序
  - 支持打开项目根目录、分类目录、文件所在目录（Windows）
- LDD 材料匹配
  - 每条 LDD 已关联材料支持填写“说明/备注”
  - 导出尽调包时同步输出 LDD_Notes.txt（包含所有说明与材料备注）
- 稳定性
  - SQLite busy/locked 处理增强（busy_timeout/WAL 等），减少“database is locked”导致的 500
  - 全局异常返回 JSON，前端错误提示更清晰
  - 去掉“导入尽调模板”入口与接口，降低误操作与不稳定因素

## 兼容与说明

- 本版本不包含本地数据库文件（.db）与备份 zip，避免上传敏感/体积文件
- 建议使用 Ctrl+F5 强制刷新以加载最新前端资源
